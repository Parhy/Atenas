PROJECT TITLE: Atenas
PURPOSE OF PROJECT: Atenas es una aplicación para introducir a los aspirantes 
de la empresa, del mismo nombre, a las actividades más importantes qué realiza la empresa.
VERSION or DATE: 23/08/2020
HOW TO START THIS PROJECT: Ejecute el método main de la clase Main.
AUTHORS: Joseph Basto, Santiago Angulo
USER INSTRUCTIONS: El usuario al ejectuar el método main tendrá acceso a 
una terminal con distintas opciones, al digitar uno de los números corrrespondiente 
a una de las opciones en la consola podrá navegar a lo largo de toda la aplicación.
